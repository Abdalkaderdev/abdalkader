## Installation

```bash
npm install @abdalkader/component-library
```

## Usage

```tsx
import { Button, Input } from '@abdalkader/component-library';

function App() {
  return (
    <div>
      <Input 
        label="Email" 
        type="email" 
        required 
      />
      <Button variant="primary" onClick={() => console.log('Clicked!')}>
        Submit
      </Button>
    </div>
  );
}
```

## Components

### Button
A versatile button component with multiple variants and sizes. Supports all standard HTML button attributes.

### Input
A fully accessible input component with label, error states, and helper text support.

## Accessibility

All components are built with accessibility in mind:
- Proper ARIA attributes
- Keyboard navigation support
- Screen reader friendly
- Focus management
- Color contrast compliance

## Theming

Components use CSS custom properties for theming. Override these variables to customize the look:

```css
:root {
  --primary-color: #007bff;
  --border-radius: 0.375rem;
  --font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'Roboto', sans-serif;
  /* ... more variables */
}
```
