# API Quick Start

Get up and running with the Abdalkader API in just a few minutes. This guide will walk you through making your first API request.

## Prerequisites

- An HTTP client (browser, `curl`, Postman, or your favorite HTTP library)
- Basic knowledge of REST APIs
- (Optional) An API key for authenticated endpoints

## Step 1: Choose Your Base URL

Select the appropriate base URL for your environment:

```bash
# Production
BASE_URL="https://abdalkader.dev/api"

# Staging
BASE_URL="https://staging.abdalkader.dev/api"

# Local Development
BASE_URL="http://localhost:3000/api"
```

## Step 3: Authenticate Your Requests

For endpoints that require authentication, you'll need an API key. Get your API key from the [API Dashboard](https://abdalkader.dev/api/dashboard).

<Tabs>
  <Tab title="JavaScript">
    ```javascript
    const API_KEY = 'your-api-key-here';
    
    const response = await fetch('https://abdalkader.dev/api/performance', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'X-API-Key': API_KEY
      },
      body: JSON.stringify({
        name: 'page_load_time',
        value: 1250,
        timestamp: Date.now(),
        url: 'https://abdalkader.dev',
        userAgent: navigator.userAgent
      })
    });
    
    const data = await response.json();
    console.log(data);
    ```
  </Tab>
  <Tab title="Python">
    ```python
    import requests

    API_KEY = 'your-api-key-here'
    
    response = requests.post(
        'https://abdalkader.dev/api/performance',
        headers={
            'Content-Type': 'application/json',
            'X-API-Key': API_KEY
        },
        json={
            'name': 'page_load_time',
            'value': 1250,
            'timestamp': int(time.time() * 1000),
            'url': 'https://abdalkader.dev',
            'userAgent': 'MyApp/1.0'
        }
    )
    
    data = response.json()
    print(data)
    ```
  </Tab>
  <Tab title="cURL">

## Step 4: Handle Errors

Always handle errors appropriately:

<Tabs>
  <Tab title="JavaScript">
    ```javascript
    try {
      const response = await fetch('https://abdalkader.dev/api/performance', {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json',
          'X-API-Key': API_KEY
        },
        body: JSON.stringify(metricData)
      });
      
      if (!response.ok) {
        const error = await response.json();
        throw new Error(error.message || 'Request failed');
      }
      
      const data = await response.json();
      console.log('Success:', data);
    } catch (error) {
      console.error('Error:', error.message);
    }
    ```
  </Tab>
  <Tab title="Python">
    ```python
    import requests

    try:
        response = requests.post(
            'https://abdalkader.dev/api/performance',
            headers={
                'Content-Type': 'application/json',
                'X-API-Key': API_KEY
            },
            json=metric_data
        )
        
        response.raise_for_status()
        data = response.json()
        print('Success:', data)
    except requests.exceptions.HTTPError as e:
        print(f'Error: {e.response.json()}')
    except Exception as e:
        print(f'Error: {str(e)}')
    ```
  </Tab>
</Tabs>

## Common Use Cases

### Health Monitoring

Check system health periodically:

```javascript
async function checkHealth() {
  const response = await fetch('https://abdalkader.dev/api/health');
  const data = await response.json();
  
  if (data.status === 'healthy') {
    console.log('System is healthy');
  } else {
    console.warn('System health check failed');
  }
}

// Check every 5 minutes
setInterval(checkHealth, 5 * 60 * 1000);
```

### Performance Tracking

Submit performance metrics:

```javascript
async function trackPerformance(metricName, value, url) {
  await fetch('https://abdalkader.dev/api/performance', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'X-API-Key': API_KEY
    },
    body: JSON.stringify({
      name: metricName,
      value: value,
      timestamp: Date.now(),
      url: url,
      userAgent: navigator.userAgent
    })
  });
}

// Track page load time
window.addEventListener('load', () => {
  const loadTime = performance.timing.loadEventEnd - performance.timing.navigationStart;
  trackPerformance('page_load_time', loadTime, window.location.href);
});
```

### Error Reporting

Report errors for debugging:

```javascript
async function reportError(error, context = {}) {
  await fetch('https://abdalkader.dev/api/errors', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'X-API-Key': API_KEY
    },
    body: JSON.stringify({
      message: error.message,
      stack: error.stack,
      url: window.location.href,
      userAgent: navigator.userAgent,
      timestamp: Date.now(),
      sessionId: getSessionId(),
      environment: 'production',
      additionalContext: context
    })
  });
}

// Global error handler
window.addEventListener('error', (event) => {
  reportError(event.error, {
    filename: event.filename,
    lineno: event.lineno,
    colno: event.colno
  });
});
```
