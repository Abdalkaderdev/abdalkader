## Overview

The Button component provides a flexible and accessible way to trigger actions in your application. It supports multiple variants, sizes, and states.

## Import

```tsx
import { Button } from '@abdalkader/component-library';
```

## Basic Usage

```tsx
<Button variant="primary" onClick={() => console.log('Clicked')}>
  Click me
</Button>
```

## Form Integration

### Submit Button

```tsx
<form onSubmit={handleSubmit}>
  <Input label="Email" type="email" />
  <Button type="submit" variant="primary">
    Submit
  </Button>
</form>
```

## Accessibility

The Button component includes proper ARIA attributes and keyboard support:

```tsx
<Button
  aria-label="Close dialog"
  aria-describedby="dialog-description"
>
  Ã—
</Button>
```

### Keyboard Support

| Key | Action |
|-----|--------|
| `Enter` | Activates the button |
| `Space` | Activates the button |

## Props

<ResponseField name="variant" type="'primary' | 'secondary' | 'danger'" default="primary">
  Visual style variant of the button
</ResponseField>

<ResponseField name="size" type="'small' | 'medium' | 'large'" default="medium">
  Size of the button
</ResponseField>

<ResponseField name="disabled" type="boolean" default="false">
  Disables the button
</ResponseField>

<ResponseField name="type" type="'button' | 'submit' | 'reset'" default="button">
  HTML button type
</ResponseField>

<ResponseField name="onClick" type="() => void">
  Click event handler
</ResponseField>

<ResponseField name="children" type="React.ReactNode" required>
  Button content
</ResponseField>
